# Student Finance Tracker

A clean, fast web app for students to track income and expenses with real-time dashboards, charts, and data export.

## Features

 **Dashboard**: Balance, income/expense overview with charts
 **Transactions**: Search, filter, and manage all transactions
 **Charts**: Cashflow line chart and category breakdown
 **Export**: Download data as CSV or JSON
 **Dark/Light Mode**: Theme toggle with persistent preference
 **Mobile Friendly**: Responsive design with sidebar menu
 **Demo Data**: Load sample transactions instantly

## Usage

1. Open `index.html` in your browser
2. Choose "Load demo data" or start fresh
3. Add transactions or explore existing data
4. Switch between Dashboard and Transactions views
5. Toggle dark mode with the moon/sun icon

## Files

- `index.html` - Main HTML structure
- `scripts/app.js` - App logic and state management
- `styles/main.css` - Core styling
- `styles/responsive.css` - Mobile responsiveness
- `test/seed.json` - Demo data
